# 更新日志

## 2024-12-19 - 根据PRD更新

### 主要变更

1. **字段类型调整**
   - 错题原题和去手写字段类型从"图片"改为"附件"
   - 引导问题、思考过程、解题思路、解题清单从"多行文本"改为"文本"
   - 删除了"题目文本"字段（不再保存到飞书表格）

2. **错误类型选项更新**
   - 选项值从"不会做"/"做错了"改为"不会"/"做错"
   - 更新了所有相关代码和文档

3. **AI模型支持**
   - 根据PRD，添加DeepSeek大语言模型支持
   - 更新配置示例，添加DeepSeek API配置
   - AI引导和反馈题生成默认使用DeepSeek

4. **代码优化**
   - 更新了飞书客户端，移除题目文本字段的保存
   - 调整了获取题目文本的逻辑（需要从其他地方获取或重新识别）
   - 更新了所有相关的文档和示例

### 注意事项

- 题目文本字段已删除，如果需要使用引导学习或生成反馈题，需要提供题目文本参数
- 建议配置DeepSeek API以获得更好的AI引导效果
- 飞书表格字段需要按照新的PRD要求创建
