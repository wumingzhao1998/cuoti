## 📝 使用步骤

### 第一步：环境配置
```bash
# 安装依赖
pip install -r requirements.txt

# 复制配置文件
cp config.example.py config.py

# 编辑config.py，填入API密钥
```

### 第二步：飞书设置
1. 在飞书开放平台创建应用
2. 创建多维表格"吴涵的错题本"
3. 配置表格字段（参考USAGE.md）
4. 更新代码中的表格ID

### 第三步：测试运行
```bash
# 准备一张错题图片
# 运行主程序
python main.py path/to/question.jpg 不会做
```

## 🐛 已知问题

1. **去手写效果有限**：当前使用基础算法，效果可能不理想，建议集成专业服务
2. **飞书表格ID硬编码**：需要在代码中替换为实际值
3. **错误处理不够完善**：部分异常情况可能需要更详细的处理
4. **图片识别准确率**：依赖豆包API，可能需要根据实际情况调整提示词
